# Smart Industrial Monitoring System (Simulation)

![C Language](https://img.shields.io/badge/Language-C-blue)  
![Status](https://img.shields.io/badge/Status-Completed-brightgreen)

## Overview
This project simulates an IoT-based industrial monitoring system using C.  
It reads simulated sensor data, checks for high values, and sends data to the cloud.

## Features
- Simulates sensor readings
- Alerts for high sensor values
- Cloud communication simulation
- Runs automatically (2 iterations for demo)

- C Programming
